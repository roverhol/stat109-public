A3. Stacked bar chart (Lecture 27)

Each bar is a species; segments are Urban vs Rural; height of a segment is the count in that cell.

(a) The total height of the Robin bar equals which quantity?
a) The number of Robins (Urban + Rural).
b) The proportion of all birds that are Robins.
c) The number of Urban birds of all species.

(b) The height of the Urban segment in the Robin bar divided by the full height of the Robin bar equals which quantity?
a) A joint proportion (that cell count ÷ total birds of all species).
b) A row proportion (Urban among Robins).
c) A column proportion (Robins among Urban birds).

### A1. Scatterplot (Lecture 26)

```{r}
#| echo: false
#| fig-cap: "Practice scatterplot for Quiz 10."
#| fig-width: 5
#| fig-height: 4
set.seed(1010)
xv <- runif(40, 1, 10)
sc <- data.frame(
  x = xv,
  y = 1.2 * xv + rnorm(40, 0, 0.9)
)
ggplot(sc, aes(x = x, y = y)) +
  geom_point(alpha = 0.85, color = "darkslateblue", size = 2.2) +
  labs(title = "Variable Y vs variable X (illustrative)", x = "X", y = "Y") +
```

### A2. Boxplots comparing two groups (Lecture 25)

```{r}
#| echo: false
#| fig-cap: "Side-by-side boxplots for two groups."
#| fig-width: 4.5
#| fig-height: 4
bp <- data.frame(
  group = rep(c("Control", "Treatment"), each = 35),
  y = c(rnorm(35, 12, 2), rnorm(35, 17, 2.2))
)
ggplot(bp, aes(x = group, y = y, fill = group)) +
  geom_boxplot(alpha = 0.85, show.legend = FALSE) +
  labs(title = "Outcome Y by treatment group (illustrative)", x = NULL, y = "Y") +
  theme_minimal(base_size = 12) +
  scale_fill_manual(values = c("Control" = "gray75", "Treatment" = "steelblue"))
```

**(a)** Which group has the **larger median**?\
a) Control\
b) Treatment\
c) They are essentially the same.

**(b)** Which group’s IQR appears to be larger?\
a) Control\
b) Treatment\
c) Hard to tell from the plot.

### A3. Stacked bar chart (Lecture 27)

```{r}
#| echo: false
#| fig-cap: "Stacked counts: habitat by species."
#| fig-width: 5
#| fig-height: 4
ct <- data.frame(
  species = rep(c("Robin", "Sparrow"), each = 2),
  habitat = rep(c("Urban", "Rural"), 2),
  n = c(30, 10, 12, 28)
)
ggplot(ct, aes(x = species, y = n, fill = habitat)) +
  geom_col(color = "white", width = 0.65) +
  labs(
    title = "Bird counts by species and habitat (illustrative)",
    x = "Species",
    y = "Count",
    fill = "Habitat"
  ) +
  theme_minimal(base_size = 12) +
  scale_fill_manual(values = c("Urban" = "tan3", "Rural" = "darkseagreen4"))
```

Each **bar** is a species; **segments** are Urban vs Rural; **height of a segment** is the count in that cell.

**(a)** The **total height** of the Robin bar equals which quantity?\
a) The number of Robins (Urban + Rural).\
b) The proportion of all birds that are Robins.\
c) The number of Urban birds of all species.

**(b)** The **height of the Urban segment** in the Robin bar divided by the **full height of the Robin bar** equals which quantity?\
a) A **joint** proportion (that cell count ÷ total birds of all species).\
b) A **row** proportion (Urban among Robins).\
c) A **column** proportion (Robins among Urban birds).
