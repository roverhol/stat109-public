STAT 109: Introductory Biostatistics - Simple linear regression (part 2)

Lecture 36

Inference in simple linear regression

If the linear model is appropriate and residuals are roughly normal with constant spread, we can do interval estimation:

  • Confidence interval for the slope beta1.
  • Confidence interval for the mean response E(Y | X = x0).
  • Prediction interval for an individual future Y at x0.

The prediction interval is wider than the mean-response confidence interval because it includes both uncertainty in the mean and individual-to-individual variation.

Learning objectives

By the end of this lecture, you should be able to:

  • explain when normal-residual assumptions support interval inference in linear regression;
  • compute and interpret a confidence interval for slope beta1;
  • compute and interpret a confidence interval for the mean response at x0;
  • compute and interpret a prediction interval for an individual response at x0;
  • explain why prediction intervals are wider than confidence intervals for means;
  • report test set MSE and test R^2 as primary predictive performance summaries.

Example 1 - Study hours and exam score (continued)

Code 
if (!requireNamespace("tidyverse", quietly = TRUE)) {
  install.packages("tidyverse", repos = "https://cloud.r-project.org")
}

library(tidyverse)

set.seed(3501)
n <- 160
exam_dat <- tibble(
  study_hours = runif(n, 1, 15),
  exam_score = 45 + 3.4 * study_hours + rnorm(n, 0, 8)
)

Training vs test performance reminder

Code 
mse <- function(y, yhat) mean((y - yhat)^2)

train_pred <- predict(fit_exam, newdata = exam_train)
test_pred  <- predict(fit_exam, newdata = exam_test)

train_mse <- mse(exam_train$exam_score, train_pred)
test_mse  <- mse(exam_test$exam_score, test_pred)

train_r2 <- 1 - sum((exam_train$exam_score - train_pred)^2) /
  sum((exam_train$exam_score - mean(exam_train$exam_score))^2)
test_r2 <- 1 - sum((exam_test$exam_score - test_pred)^2) /
  sum((exam_test$exam_score - mean(exam_test$exam_score))^2)

tibble(
  set = c("train", "test"),
  MSE = c(train_mse, test_mse),
  R2 = c(train_r2, test_r2)
)
# A tibble: 2 × 3
  set     MSE    R2
  <chr> <dbl> <dbl>
1 train  43.7 0.830
2 test   66.3 0.767

Use test metrics as your main generalization report:

  • Test MSE = typical squared prediction error on new cases.
  • Test R^2 = proportion of test-set variation explained by the linear model.

Key takeaways

  • Slope CI answers: what plausible values for the linear effect beta1 are supported by data?
  • Mean-response CI answers: where is the average outcome for all units at x0?
  • Prediction interval answers: where might one future individual outcome fall at x0?
  • Keep reporting test MSE and test R^2 for predictive performance on new data.

## Inference in simple linear regression

-   Confidence interval for the slope `beta1`.
-   Confidence interval for the mean response `E(Y | X = x0)`.
-   Prediction interval for an individual future `Y` at `x0`.

The prediction interval is wider than the mean-response confidence interval because it includes both uncertainty in the mean and individual-to-individual variation.

### Learning objectives

By the end of this lecture, you should be able to:

-   explain when normal-residual assumptions support interval inference in linear regression;
-   compute and interpret a confidence interval for slope `beta1`;
-   compute and interpret a confidence interval for the mean response at `x0`;
-   compute and interpret a prediction interval for an individual response at `x0`;
-   explain why prediction intervals are wider than confidence intervals for means;
-   report test set MSE and test `R^2` as primary predictive performance summaries.
