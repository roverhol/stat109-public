STAT 109: Introductory Biostatistics — Chi-square test of independence

Lecture 33

Chi-square test of independence for two categorical variables

When both the explanatory and response variables are categorical, we summarize the data in a contingency table of counts (each cell = number of units with that combination of levels). The Pearson chi-square test of independence asks whether the data are compatible with independence in the population:

  • \(H_0\): the two variables are independent (knowing one does not change the distribution of the other).
  • \(H_a\): the two variables are not independent (there is association).

In R, build a table of counts (with table() or count() + pivot_wider()), then run chisq.test(tab). Inspect chisq.test(tab)$expected for the expected counts under independence.

Conditions.

  • Independent observations (e.g. one row per person, or clearly independent trials as designed).
  • For the usual Pearson \(p\)-value, all expected cell counts should be at least about 5. If not, use simulate.p.value = TRUE in chisq.test, collect more data, or collapse sparse categories.

Plots. Stacked bar charts (counts or 100% filled)

Example 1 — RCT: three vaccine arms vs flu (yes/no)

Story. Healthy adults were randomly assigned to one of three study arms (Placebo, Low-dose vaccine, High-dose vaccine). Over winter follow-up, whether each person had influenza (yes/no) was recorded.

Research Question: the chi-square test addresses whether flu status is independent of arm in the population of people like those enrolled (equivalently, whether flu rates differ across arms).

Code 
if (!requireNamespace("tidyverse", quietly = TRUE)) {
  install.packages("tidyverse", repos = "https://cloud.r-project.org")
}

library(tidyverse)

set.seed(3301)
n_per <- 55
p_flu <- c(Placebo = 0.22, `Low dose` = 0.12, `High dose` = 0.08)
flu_rct <- tibble(
  arm = factor(
    sample(rep(names(p_flu), each = n_per)),
    levels = c("Placebo", "Low dose", "High dose")
  )
) |>
  mutate(
    flu = factor(
      rbinom(n(), 1, p_flu[as.character(arm)]),
      levels = c(0, 1),
      labels = c("No", "Yes")
    )
  )

Interpretation. If expected counts are fine and the p-value is small, the sample is not consistent with independence between arm and flu; describe which arms have higher observed flu fractions using the table or plot.

# Lecture 33

## Chi-square test of independence for two categorical variables

When both the explanatory and response variables are categorical, we summarize the data in a contingency table of counts (each cell = number of units with that combination of levels). The Pearson chi-square test of independence asks whether the data are compatible with independence in the population:

-   $H_0$: the two variables are independent (knowing one does not change the distribution of the other).
-   $H_a$: the two variables are not independent (there is association).

In R, build a table of counts (with `table()` or `count()` + `pivot_wider()`), then run **`chisq.test(tab)`**. Inspect **`chisq.test(tab)$expected`** for the **expected counts** under independence.

**Conditions.**

-   **I**ndependent observations (e.g. one row per person, or clearly independent trials as designed).
-   For the usual Pearson $p$-value, all expected cell counts should be at least about 5. If not, use `simulate.p.value = TRUE` in `chisq.test`, collect more data, or collapse sparse categories.

**Plots.** **Stacked bar charts** (counts or **100%** filled)

------------------------------------------------------------------------

## Example 1 — RCT: three vaccine arms vs flu (yes/no)

**Story.** Healthy adults were **randomly assigned** to one of **three** study arms (**Placebo**, **Low-dose vaccine**, **High-dose vaccine**). Over winter follow-up, whether each person had **influenza (yes/no)** was recorded.

Research Question: the chi-square test addresses whether **flu status** is **independent** of **arm** in the population of people like those enrolled (equivalently, whether **flu rates** differ across arms).

```{r}
if (!requireNamespace("tidyverse", quietly = TRUE)) {
  install.packages("tidyverse", repos = "https://cloud.r-project.org")
}

library(tidyverse)

set.seed(3301)
n_per <- 55
p_flu <- c(Placebo = 0.22, `Low dose` = 0.12, `High dose` = 0.08)
flu_rct <- tibble(
  arm = factor(
    sample(rep(names(p_flu), each = n_per)),
    levels = c("Placebo", "Low dose", "High dose")
  )
) |>
  mutate(
    flu = factor(
      rbinom(n(), 1, p_flu[as.character(arm)]),
      levels = c(0, 1),
      labels = c("No", "Yes")
    )
  )
```

### Table of counts

```{r}
flu_rct |>
  count(arm, flu)
```

### Counts in wide form (rows = arm, columns = flu)

```{r}
wide_counts <- flu_rct |>
  count(arm, flu) |>
  pivot_wider(names_from = flu, values_from = n, values_fill = 0)
wide_counts
```

### Row proportions (flu rate within each arm)

```{r}
flu_rct |>
  count(arm, flu) |>
  group_by(arm) |>
  mutate(prop_within_arm = n / sum(n)) |>
  ungroup()
```

### 100% stacked bar chart (proportion with/without flu, by arm)

```{r}
flu_rct |>
  count(arm, flu) |>
  ggplot(aes(x = arm, y = n, fill = flu)) +
  geom_col(position = "fill") +
  scale_y_continuous(labels = scales::percent_format()) +
  labs(
    title = "Flu (yes/no) distribution by study arm (100% stacked)",
    x = "Study arm",
    y = "Proportion within arm",
    fill = "Flu"
  ) +
  theme_minimal()
```

### Chi-square test of independence

```{r}
tab_flu <- table(Arm = flu_rct$arm, Flu = flu_rct$flu)
tab_flu
chisq.test(tab_flu)$expected
chisq.test(tab_flu)
```

**Interpretation.** If expected counts are fine and the *p*-value is small, the sample is **not** consistent with **independence** between arm and flu; describe **which arms** have higher **observed** flu fractions using the table or plot.

## Example 2 

**Story**. Shoppers at a campus store were randomly shown one of three package designs (D1, D2, D3) for the same snack; whether they purchased (yes/no) was recorded.

```{r}
set.seed(3302)
n_per <- 48

    purchase = factor(
      rbinom(n(), 1, p_buy[as.character(design)]),
```

### Counts and column proportions (purchase rate by design)

```{r}
pack_rct |>

pack_rct |>
```
