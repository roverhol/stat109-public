STAT 109: Introductory Biostatistics

Lecture 27: Two categorical variables — contingency tables, proportions, and stacked bar charts

Learning objectives

  • Build a two-way table (contingency table) of counts for two categorical variables.
  • Convert counts to proportions three ways: joint (of the grand total), within rows, and within columns—and say which question each answers.
  • Make stacked bar charts in ggplot2 (geom_col() with fill); interpret stacked counts vs 100% stacked (position = "fill").
  • Reproduce tables and plots using the tidyverse (count(), group_by(), mutate(), pivot_wider()).

Why two categorical variables together

Each observation has a level of variable A (e.g. species) and a level of variable B (e.g. habitat). We summarize how the combinations co-occur: counts in each cell of a table, then optional graphs that show those counts or proportions at a glance.

Small example by hand (two binary variables)

Setting. Birds counted in two habitats: Urban or Rural. Species recorded as Robin or Sparrow (ignore other species here).

Counts (made-up, small enough to add on paper):

Urban Rural Row total
Robin 8 2 10
Sparrow 4 6 10
Column total 12 8 20 = grand total \(N\)

So there are 20 birds. The joint count in each cell is a count of birds with that (species, habitat) pair.

Joint proportions (of the grand total)

Each cell count divided by \(N = 20\) answers: What fraction of all birds fall in this cell?

  • Robin & Urban: \(8/20 = 0.40\)
  • Robin & Rural: \(2/20 = 0.10\)
  • Sparrow & Urban: \(4/20 = 0.20\)
  • Sparrow & Rural: \(6/20 = 0.30\)

Check: \(0.40 + 0.10 + 0.20 + 0.30 = 1\).

Row proportions (conditional on species)

Within each species, divide by that row’s total (10). Answers: Among Robins only, what fraction were Urban vs Rural? (and similarly for Sparrows.)

Robin: Urban \(8/10 = 0.80\), Rural \(2/10 = 0.20\).

Sparrow: Urban \(4/10 = 0.40\), Rural \(6/10 = 0.60\).

Each row sums to \(1\).

Column proportions (conditional on habitat)

Within each habitat, divide by that column’s total. Answers: Among Urban birds, what fraction were Robin vs Sparrow?

Urban: Robin \(8/12 \approx 0.67\), Sparrow \(4/12 \approx 0.33\) (column total 12).

Rural: Robin \(2/8 = 0.25\), Sparrow \(6/8 = 0.75\) (column total 8).

Each column sums to \(1\).

Takeaway. The same four counts support three different stories depending on whether you normalize by everything (joint), by rows (species), or by columns (habitat). Be explicit about which denominator you use when you write a sentence or a plot.

Example 1 — The bird table in R (tidyverse)

We store one row per bird: variables species and habitat.

Code 
if (!requireNamespace("tidyverse", quietly = TRUE)) {
  install.packages("tidyverse", repos = "https://cloud.r-project.org")
}

library(tidyverse)

birds <- tibble(
  species = c(rep("Robin", 10), rep("Sparrow", 10)),
  habitat = c(rep(c("Urban", "Rural"), c(8, 2)), rep(c("Urban", "Rural"), c(4, 6)))
) |>
  mutate(
    species = factor(species, levels = c("Robin", "Sparrow")),
    habitat = factor(habitat, levels = c("Urban", "Rural"))
  )

Two-way table of counts

Code 
birds |>
  count(species, habitat) |>
  pivot_wider(names_from = habitat, values_from = n, values_fill = 0)
# A tibble: 2 × 3
  species Urban Rural
  <fct>   <int> <int>
1 Robin       8     2
2 Sparrow     4     6

Joint, row, and column proportions (long format)

Code 
joint <- birds |>
  count(species, habitat) |>
  mutate(prop_joint = n / sum(n))

row_prop <- birds |>
  count(species, habitat) |>
  group_by(species) |>
  mutate(prop_row = n / sum(n)) |>
  ungroup()

col_prop <- birds |>
  count(species, habitat) |>
  group_by(habitat) |>
  mutate(prop_col = n / sum(n)) |>
  ungroup()

Stacked bar chart — counts

Bars show species; segments show habitat; height of each segment is the cell count.

Code 
birds |>
  count(species, habitat) |>
  ggplot(aes(x = species, y = n, fill = habitat)) +
  geom_col(color = "white") +
  labs(
    title = "Bird counts by species and habitat",
    x = NULL,
    y = "Count",
    fill = "Habitat"
  )

Stacked bar chart — 100% within each species (row proportions)

position = "fill" stretches each bar to height 1; segment heights match row proportions (prop_row). Good for comparing habitat mix between species when raw counts differ.

Code 
birds |>
  count(species, habitat) |>
  ggplot(aes(x = species, y = n, fill = habitat)) +
  geom_col(position = "fill", color = "white") +
  scale_y_continuous(labels = scales::label_percent()) +
  labs(
    title = "Habitat distribution within each species",
    x = NULL,
    y = "Proportion (within species)",
    fill = "Habitat"
  )

Optional. Swap what is on the x-axis and what is fill if the research question reads more naturally the other way (e.g. habitat on the x-axis, species as fill).

Summary

For two categorical variables, start with counts in a two-way table, then compute proportions with a clear denominator: joint (\(N\) everywhere), row (within levels of the variable on the rows), or column (within levels of the variable on the columns). # Lecture 27: Two categorical variables — contingency tables, proportions, and stacked bar charts

## Learning objectives

-   Build a **two-way table** (contingency table) of **counts** for two categorical variables.
-   Convert counts to **proportions** three ways: **joint** (of the grand total), **within rows**, and **within columns**—and say which question each answers.
-   Make **stacked bar charts** in **ggplot2** (`geom_col()` with `fill`); interpret **stacked counts** vs **100% stacked** (`position = "fill"`).
-   Reproduce tables and plots using the **tidyverse** (`count()`, `group_by()`, `mutate()`, `pivot_wider()`).

------------------------------------------------------------------------

## Why two categorical variables together

Each observation has a level of variable **A** (e.g. species) and a level of variable **B** (e.g. habitat). We summarize how the combinations **co-occur**: counts in each **cell** of a table, then optional **graphs** that show those counts or **proportions** at a glance.

------------------------------------------------------------------------

## Small example by hand (two binary variables)

**Setting.** Birds counted in two habitats: **Urban** or **Rural**. Species recorded as **Robin** or **Sparrow** (ignore other species here).

**Counts** (made-up, small enough to add on paper):

|           | Urban | Rural | **Row total** |
|-----------|-------|-------|---------------|
| **Robin**   | 8     | 2     | 10            |
| **Sparrow** | 4     | 6     | 10            |
| **Column total** | 12    | 8     | **20** = grand total $N$ |

So there are **20** birds. The **joint** count in each cell is a count of birds with that **(species, habitat)** pair.

### Joint proportions (of the grand total)

Each cell count divided by $N = 20$ answers: *What fraction of **all** birds fall in this cell?*

-   Robin & Urban: $8/20 = 0.40$
-   Robin & Rural: $2/20 = 0.10$
-   Sparrow & Urban: $4/20 = 0.20$
-   Sparrow & Rural: $6/20 = 0.30$

Check: $0.40 + 0.10 + 0.20 + 0.30 = 1$.

### Row proportions (conditional on species)

Within each **species**, divide by that row’s total (10). Answers: *Among Robins only, what fraction were Urban vs Rural?* (and similarly for Sparrows.)

**Robin:** Urban $8/10 = 0.80$, Rural $2/10 = 0.20$.

**Sparrow:** Urban $4/10 = 0.40$, Rural $6/10 = 0.60$.

Each row sums to $1$.

### Column proportions (conditional on habitat)

Within each **habitat**, divide by that column’s total. Answers: *Among Urban birds, what fraction were Robin vs Sparrow?*

**Urban:** Robin $8/12 \approx 0.67$, Sparrow $4/12 \approx 0.33$ (column total 12).

**Rural:** Robin $2/8 = 0.25$, Sparrow $6/8 = 0.75$ (column total 8).

Each column sums to $1$.

**Takeaway.** The **same four counts** support three different stories depending on whether you normalize by **everything** (joint), by **rows** (species), or by **columns** (habitat). Be explicit about which denominator you use when you write a sentence or a plot.

## Example 1 — The bird table in R (tidyverse)

We store one row per bird: variables `species` and `habitat`.

```{r}
if (!requireNamespace("tidyverse", quietly = TRUE)) {
  install.packages("tidyverse", repos = "https://cloud.r-project.org")
}

library(tidyverse)

birds <- tibble(
  species = c(rep("Robin", 10), rep("Sparrow", 10)),
  habitat = c(rep(c("Urban", "Rural"), c(8, 2)), rep(c("Urban", "Rural"), c(4, 6)))
) |>
  mutate(
    species = factor(species, levels = c("Robin", "Sparrow")),
    habitat = factor(habitat, levels = c("Urban", "Rural"))
  )
```

### Two-way table of counts

```{r}
birds |>
  count(species, habitat) |>
  pivot_wider(names_from = habitat, values_from = n, values_fill = 0)
```

### Joint, row, and column proportions (long format)

```{r}
joint <- birds |>
  count(species, habitat) |>
  mutate(prop_joint = n / sum(n))

row_prop <- birds |>
  count(species, habitat) |>
  group_by(species) |>
  mutate(prop_row = n / sum(n)) |>
  ungroup()

col_prop <- birds |>
  count(species, habitat) |>
  group_by(habitat) |>
  mutate(prop_col = n / sum(n)) |>
  ungroup()

joint
row_prop
col_prop
```

### Stacked bar chart — counts

Bars show **species**; segments show **habitat**; height of each segment is the **cell count**.

```{r}
birds |>
  count(species, habitat) |>
  ggplot(aes(x = species, y = n, fill = habitat)) +
  geom_col(color = "white") +
  labs(
    title = "Bird counts by species and habitat",
    x = NULL,
    y = "Count",
    fill = "Habitat"
  )
```

### Stacked bar chart — 100% within each species (row proportions)

`position = "fill"` stretches each bar to height 1; segment **heights** match **row proportions** (`prop_row`). Good for comparing **habitat mix** between species when raw counts differ.

```{r}
birds |>
  count(species, habitat) |>
  ggplot(aes(x = species, y = n, fill = habitat)) +
  geom_col(position = "fill", color = "white") +
  scale_y_continuous(labels = scales::label_percent()) +
  labs(
    title = "Habitat distribution within each species",
    x = NULL,
    y = "Proportion (within species)",
    fill = "Habitat"
  )
```

*Optional.* Swap what is on the **x-axis** and what is **`fill`** if the research question reads more naturally the other way (e.g. habitat on the x-axis, species as fill).

## Example 2 — `mpg`: `class` vs `drv` (several categories each)

Each row is a car model. We cross **vehicle class** and **drive type**.

```{r}
data("mpg", package = "ggplot2")

mpg |>
  count(class, drv) |>
  pivot_wider(names_from = drv, values_from = n, values_fill = 0)
```

Row proportions: among cars in a given **class**, what fraction have each **drv**?

```{r}
mpg |>
  count(class, drv) |>
  group_by(class) |>
  mutate(prop_within_class = n / sum(n)) |>
  ungroup() |>
  print(n = 20)
```

Stacked bars (counts and 100% stacked):

```{r}
mpg_ct <- mpg |> count(class, drv)

ggplot(mpg_ct, aes(x = class, y = n, fill = drv)) +
  geom_col(color = "white") +
  labs(
    title = "Drive type by vehicle class (counts)",
    x = "Class",
    y = "Count",
    fill = "Drive type"
  ) +
  theme(axis.text.x = element_text(angle = 35, hjust = 1))

ggplot(mpg_ct, aes(x = class, y = n, fill = drv)) +
  geom_col(position = "fill", color = "white") +
  scale_y_continuous(labels = scales::label_percent()) +
  labs(
    title = "Drive type mix within class (row proportions)",
    x = "Class",
    y = "Proportion within class",
    fill = "Drive type"
  ) +
  theme(axis.text.x = element_text(angle = 35, hjust = 1))
```

## Example 3 — TidyTuesday Palmer Penguins: `species` vs `island`

We load the published TidyTuesday file from GitHub with `read_csv()`. (If you are offline, download the CSV once and use a local path instead.)

```{r}
penguins <- read_csv(

penguins |>
```

## Summary

For two **categorical** variables, start with **counts** in a **two-way table**, then compute **proportions** with a clear **denominator**: **joint** ($N$ everywhere), **row** (within levels of the variable on the rows), or **column** (within levels of the variable on the columns). **Stacked bar charts** display the same structure: **stacked counts** show **how many**; **position = "fill"** shows **fractions within** each bar (match the fraction definition to **rows** or **columns** by choosing **x** and **fill** thoughtfully).
