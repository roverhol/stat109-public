How to use this lab

  1. Demo notebook (lab-8-demo.ipynb) — follow along in Thursday lab using mpg (highway MPG, drive type, vehicle class, cylinders).
  2. Student notebook (lab-8-student.ipynb) — complete five questions on diamonds and turn in on Canvas by Tuesday (see Assessments).

Learning outcomes

By the end of this lab, you should be able to:

  1. Load a tidyverse dataset and inspect variables with glimpse().
  2. Identify whether a variable is numerical or categorical (class(), levels()).
  3. Recode a variable when numeric values are really category codes (factor(), mutate()).
  4. For one numerical variable: compute mean and SD and make a histogram.
  5. For one categorical variable: compute count and proportion by level.
  6. Use filter() to subset rows with conditions.
  7. Use group_by() with summarise() to compare groups (mean, SD, n()).

Function reference

Function What it does
library(tidyverse) Loads core tidyverse packages
glimpse(df) Quick view of names and types
class(x), levels(x) Type and category levels
mutate(), factor() Create or recode variables
mean(), sd() Center and spread for numerical data
count(), summarise(), n() Counts and grouped summaries
filter() Keep rows matching conditions
group_by() Define groups before summarizing
ggplot() + geom_histogram() Histogram for numerical data
as.numeric(), as.factor() Convert between types

## Learning outcomes

By the end of this lab, you should be able to:

1. Load a tidyverse dataset and inspect variables with `glimpse()`.
2. Identify whether a variable is **numerical** or **categorical** (`class()`, `levels()`).
3. Recode a variable when numeric values are really **category codes** (`factor()`, `mutate()`).
4. For one **numerical** variable: compute **mean** and **SD** and make a **histogram**.
5. For one **categorical** variable: compute **count** and **proportion** by level.
6. Use **`filter()`** to subset rows with conditions.
7. Use **`group_by()`** with **`summarise()`** to compare groups (mean, SD, `n()`).
