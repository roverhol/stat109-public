Learning outcomes

By the end of this lab, you should be able to:

  1. Compute factorials, permutations, and combinations in R (factorial(), prod(), choose()).
  2. Identify \(n\), \(p\), and \(x\) for a binomial count and compute \(P(X = x)\) with dbinom().
  3. Use pbinom() for cumulative and tail probabilities (watch lower.tail and the cutoff).
  4. Simulate binomial outcomes with rbinom() and with a for loop + sample().
  5. Compare simulated proportions to exact binomial probabilities on a bar plot (geom_bar() with after_stat(prop) vs geom_col() with dbinom).
  6. Mark observed data on a binomial plot with geom_vline() and interpret a tail-area p-value.
  7. Estimate \(P(D \mid T^+)\) by simulation and check with Bayes’ rule.

New R functions (since Labs 1–3)

Counting / combinatorics

  • factorial(n) — compute \(n!\)
  • prod(1:n) — another way to compute \(n!\)
  • choose(n, k) — combinations \({}_nC_k\)

Binomial distribution

  • dbinom(x, size = n, prob = p) — \(P(X = x)\)
  • pbinom(q, size = n, prob = p) — \(P(X \leq q)\) (adjust lower.tail for upper tails)
  • rbinom(n_sims, size = n, prob = p) — simulate counts of successes in \(n\) trials

Plotting (ggplot2)

  • ggplot(), aes() — map variables to axes
  • geom_bar() with after_stat(count) or after_stat(prop) — bar charts from raw data
  • geom_col() — bars from precomputed heights (e.g. dbinom probabilities)
  • geom_vline(xintercept = ...) — vertical reference line at observed \(x\)

## New R functions (since Labs 1–3)

### Counting / combinatorics

- `factorial(n)` — compute $n!$
- `prod(1:n)` — another way to compute $n!$
- `choose(n, k)` — combinations ${}_nC_k$

### Binomial distribution

- `dbinom(x, size = n, prob = p)` — $P(X = x)$
- `pbinom(q, size = n, prob = p)` — $P(X \leq q)$ (adjust `lower.tail` for upper tails)
- `rbinom(n_sims, size = n, prob = p)` — simulate counts of successes in $n$ trials

### Plotting (`ggplot2`)

- `ggplot()`, `aes()` — map variables to axes
- `geom_bar()` with `after_stat(count)` or `after_stat(prop)` — bar charts from raw data
- `geom_col()` — bars from precomputed heights (e.g. `dbinom` probabilities)
- `geom_vline(xintercept = ...)` — vertical reference line at observed $x$
